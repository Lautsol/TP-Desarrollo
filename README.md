# Sistema de Gestión de Pedidos
Proyecto backend desarrollado en **Java** como trabajo práctico, orientado a la gestión de pedidos, clientes y productos.  
El sistema implementa lógica de negocio, persistencia de datos y buenas prácticas de programación orientada a objetos.

---

##  Descripción

El **Sistema de Gestión de Pedidos** permite administrar pedidos realizados por clientes, manejar productos, calcular totales y gestionar el estado de los pedidos.  
Está pensado como una aplicación backend, aplicando conceptos fundamentales de desarrollo de software.

---

##  Tecnologías utilizadas

- **Java**
- **Spring Boot**
- **Base de datos relacional (SQL)**
- **Git & GitHub**
- **Programación Orientada a Objetos (POO)**

---

##  Arquitectura

El proyecto sigue una **arquitectura en capas**, separando responsabilidades:

- **Controladores**: manejo de solicitudes
- **Servicios**: lógica de negocio
- **Repositorios**: acceso a datos
- **Modelos / Entidades**: representación del dominio

---

##  Funcionalidades principales

- Gestión de **clientes**
- Gestión de **productos**
- Creación y administración de **pedidos**
- Operaciones **CRUD**
- Cálculo de totales de pedidos
- Manejo de estados de pedidos
- Validaciones y control de errores
